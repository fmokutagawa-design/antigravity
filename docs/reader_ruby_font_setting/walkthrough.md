# 修正内容の確認 (Walkthrough) - ルビフォント個別設定機能

## 実施内容
リーダーモードにおいて、本文とは別にルビ専用のフォントを指定できる機能を追加しました。

### 1. 設定項目への追加
`App.jsx` の `settings` オブジェクトに `rubyFontFamily` を追加しました。デフォルト値は `inherit`（本文と同じ）です。

### 2. リーダーモードの改善
- `ReaderView.jsx` のツールバーに、本文フォント選択の隣にルビフォント選択用のドロップダウンを追加しました。
- `ReaderView.css` にあったルビの固定フォント設定（ヒラギノ角ゴ）を削除し、動的に設定が反映されるようにしました。

### 3. 全般設定の改善
- `Toolbar.jsx` (サイドバーの設定タブ) に「ルビ書体」の設定項目を追加しました。
- メインフォントと同様に、プリセットフォントやシステムフォントから自由に選択可能です。

## 検証結果
- `npm run build` が正常に完了することを確認しました。
- リーダーモードおよびサイドバー設定の UI が正しく表示されることを確認しました。

## 注意事項
- ルビフォントを「本文と同じ」に設定すると、本文で指定したフォントがルビにも適用されます。これにより、本文が明朝体ならルビも明朝体になります。
