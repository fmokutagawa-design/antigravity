## 提案される変更

### Editor コンポーネントの最適化 (src/components/Editor.jsx)

#### [MODIFY] [Editor.jsx](file:///Volumes/Black6T/Nexus_Dev/antigravity/src/components/Editor.jsx)

1. **無限ループ防止ガードの実装**
   - `isProcessingPropValueRef` を使用して、`useEffect` 内での `value` 同期中に `onChange` が呼ばれないようにガードします。

2. **`slideWindow` の精度向上**
   - 全文検索からのジャンプ時、ジャンプ先の位置が窓の端にならないよう、可能な限り中央（HALF）に来るようにオフセット計算を微調整します。
   - ジャンプ後に `textarea` へフォーカスを当てる処理を確実に実行します。

3. **アンダーレイ描画のチラつき防止**
   - `debouncedValue` への反映タイミングと `textarea.value` の更新をより密に同期させます。
   - `scrollForce` による再計算トリガーの条件を整理します。

4. **コードのクリーンアップ**
   - 定義されているが使用されていない `lastNotifiedValueRef` などの変数を削除します。
   - `paragraphIndex` 内の `splitString` 呼び出しを最適化し、ループ内でのオーバーヘッドを削減します。
