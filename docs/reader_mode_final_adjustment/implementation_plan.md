## 確認計画

### 手動確認
1.  **ボタン表示**: フッターのボタンが「📖 リーダー」になっていることを確認。
2.  **基本動作**: Alt+R またはボタンクリックでリーダーが開き、Escで閉じることを確認。
3.  **ツールバー**: 上部にツールバーが表示され、フォント・サイズ・縦横・テーマの切り替えが機能することを確認。
4.  **放置非表示**: 3秒間マウス操作がない場合にツールバーが隠れ、動かすと再表示されることを確認。
5.  **目次（TOC）**: 見出し（■等）がある場合に目次ボタンが表示され、ジャンプが機能することを確認。
6.  **表示品質**: 禁則処理（句読点の行頭禁止など）や、段落間の空行が正しく表示されていることを確認。
