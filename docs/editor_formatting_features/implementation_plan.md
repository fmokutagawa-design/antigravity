# 実装計画：エディタ文字整形機能の強化と不要機能の整理

AIが過剰に付与したルビを効率的に掃除するため、エディタの選択範囲に対して「ルビ削除」を実行できる機能を追加します。また、独自タグ形式で機能していなかった「フォント変更」機能を削除し、エディタをスッキリさせます。

## Proposed Changes

### [Component: Utils]

#### [MODIFY] [formatText.js](file:///Volumes/Black6T/Nexus_Dev/antigravity/src/utils/formatText.js)
- `applyFormat` 関数に加えて、純粋なテキスト変換のみを行う `cleanRuby(text)` などのユーティリティ関数をエクスポートし、エディタから直接利用できるようにします。
- AA整形にも役立つ `convertFullWidth(text)` も公開します。

### [Component: Editor]

#### [MODIFY] [Editor.jsx](file:///Volumes/Black6T/Nexus_Dev/antigravity/src/components/Editor.jsx)
- **シンタックスハイライト**:
    - `patterns` から `{font:...}` 記法（`type: 'font'`）の正規表現を削除。
- **コンテキストメニュー**:
    - 「🔤 フォント変更 ▸」およびそのサブメニュー、関連する解除ロジックをすべて削除。
    - 新しく「✨ 文字整形 ▸」サブメニューを追加。以下の項目を実装：
        - **「ルビを削除」**: 選択範囲内の `｜漢字《るび》` または `漢字《るび》` を漢字のみにします。
        - **「半角を全角に」**: 選択範囲内の半角英数字・記号を全角に変換します。
        - **「空行を整理」**: 選択範囲内の連続する空行を1行にまとめます。
- **アクション実装**:
    - 各整形アクション実行後、`pushHistory` で Undo 履歴に保存し、`applyText` で DOM を更新、`onChange` で親コンポーネントへ通知します。

## 期待される効果
- AIが過剰に振ったルビを、ガバッと選択して一瞬で消去できるようになります。
- 独自タグによる将来的なトラブル（印刷時のノイズ等）を未然に防ぎます。
- エディタのコンテキストメニューが実用的でスッキリしたものになります。

## Verification Plan
### Manual Verification
1. 適当な文章（ルビ付き）を選択し、右クリックメニューから「文字整形 > ルビを削除」を実行。ルビのみが消えることを確認。
2. 半角文字が含まれる範囲を選択し、「文字整形 > 半角を全角に」を実行。全角に変換されることを確認。
3. 変更後に Undo (Cmd+Z) を行い、元に戻ることを確認。
4. 旧「フォント変更」タグがハイライトされなくなっていることを確認。
