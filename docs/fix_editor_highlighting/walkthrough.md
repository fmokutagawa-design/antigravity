# エディタ着色（シンタックスハイライト）の復旧完了

大規模文書（14万字等）でエディタ内の文字着色が消えていた問題を修正しました。

## 実施内容

- **ハイライト計算の文字数制限の緩和**:
    - `Editor.jsx` 内の `HIGHLIGHT_CHAR_LIMIT` を **20,000文字から200,000文字** に引き上げました。
    - これにより、14万字の文書でも正規表現によるマッチングが実行されるようになります。

- **パフォーマンスの担保**:
    - 前回の修正で導入した「実際に描画されるDOM要素数を2,000件に絞る（`MAX_HIGHLIGHT_ELEMENTS`）」処理は維持しています。
    - そのため、計算対象が増えてもブラウザの描画負荷（DOMノード数）が爆発することはありません。
    - ハイライト計算はデバウンス（入力停止後300ms）後に行われるため、タイピングのレスポンスへの影響も最小限に抑えられています。
