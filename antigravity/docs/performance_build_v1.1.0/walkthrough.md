# 最新ビルド報告 (v1.1.0)

42万字クラスの大規模原稿におけるパフォーマンス問題を解消するための、以下の最適化をすべて統合した最新ビルドを作成しました。

## 統合された修正内容

| カテゴリ | 修正内容 | 改善効果 |
| :--- | :--- | :--- |
| **表示ロジック** | 縦書き約物変換に `RegExp.test` による早期 return を導入 | 変換不要な段落での処理コストをほぼゼロに削減 |
| **永続化レイヤー** | 設定保存 (`app:saveSettings`) の非同期化 | 設定変更時のメインプロセス・ブロックを解消 |
| **メモリ/I/O** | 10万字超のファイルでの `localStorage` 全文保存をスキップ | 打鍵時およびオートセーブ時のレンダラー・フリーズを回避 |
| **基盤モデル** | 段落ベースの高性能ドキュメントモデルと Worker バッチング | 全文座標計算の負荷を分散・キャッシュ化 |

## ビルド情報

- **バージョン**: 1.1.0
- **ブランチ**: `perf/compose-batch-fix` (最新の `main` 最適化をマージ済み)
- **ビルド成果物**: [Antigravity-1.1.0.dmg](file:///Volumes/Black6T/Nexus_Dev/antigravity/release/Antigravity-1.1.0.dmg) (約116MB)
