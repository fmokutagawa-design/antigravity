# ルビ変換機能の拡張（全角英数字への対応）

ユーザーの指摘に基づき、「ＡＭＢＡＣ（アンバック）」のような「全角英数字＋括弧」のケースもルビ記法の変換対象に含めます。

## 変更内容

### 1. [formatText.js](file:///Volumes/Black6T/Nexus_Dev/antigravity/src/utils/formatText.js) の正規表現更新
`applyFormat` 関数の `type === 'ruby'` 部分の正規表現を以下のように変更します。

- **変更前**: `([一-龠々〆ヵヶ]+)[（(]([ぁ-んァ-ヶー]+)[）)]`
- **変更後**: `([一-龠々〆ヵヶＡ-Ｚａ-ｚ０-９]+)[（(]([ぁ-んァ-ヶー]+)[）)]`
  - 第1グループに **全角英数字**（`Ａ-Ｚａ-ｚ０-９`）を追加しました。
  - カタカナやひらがなのルビが直後にある全角英字表記も変換対象となります。

---

## 検証プラン
1. **追加テストケース**:
   - 入力: `ＡＭＢＡＣ（アンバック）`
   - 期待値: `ＡＭＢＡＣ《アンバック》`
2. **既存テストケースの維持**:
   - 入力: `主機（メイン）`
   - 期待値: `主機《メイン》`
3. **除外対象（維持）**:
   - 入力: `Main(メイン)`
   - 期待値: `Main(メイン)`（半角アルファベットは依然として除外）
